# Embedded Systems Laboratory

This repository contains my Embedded Systems laboratory projects using Arduino and Python.

## Labs

### Lab 1 – LED Sequence
Simple program controlling multiple LEDs sequentially.

### Lab 2 – Joystick Direction Detection
Reads analog joystick input and lights LEDs based on direction.

### Lab 3 – Reaction Game with RTC
Uses a DS1307 real-time clock and MAX7219 LED matrix to create a reaction timing game.

### Lab 4 – Joystick Monitoring GUI
Arduino sends joystick data to a Python PyQt6 interface for real-time visualization.

### Lab 5 – Sound Monitoring System
Electret microphone module measures sound levels.  
Data is displayed on an LCD and transmitted to a Python interface.

## Technologies

- Arduino (C/C++)
- Python
- PyQt6
- Serial Communication
- Interrupts
- ADC

## Hardware Used

- Arduino Uno
- Electret microphone module
- Joystick module
- MAX7219 LED matrix
- LCD 16x2
- LEDs
